CSS-BY-JS

The next step of CSS-in-JS.

All of the CSS you see on this page started with a standard CSS file, which was then processed into an array of rules, and then passed into a JS function that applied it to the appropriate elements on the page.

Go ahead. Inspect the page. The only CSS file that's loaded is from Google Fonts.

Why?

It's real hot to do weird things to use CSS these days.
